# Medical-Device
自制物联网终端设备，采用双主控方案，分别是UI板（上板）和传感器板（下板），烧录代码需要注意，下板的代码只能烧录给下板，上板同理。
工程包含了MATLAB上位机文件和手机APP，手机APP可以使用麻省理工大学的APP inventor软件进行二次开发，多个工程以最新的为准。

UI板是购买的淘宝上的TouchGFX开发板，价格100多，链接：
